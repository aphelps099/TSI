## How to Download
You can download these manually from your browser, or use these commands if you have direct access:

```bash
# Logos
curl -o assets/logos/technisoil-logo.svg "https://neopave.com/wp-content/uploads/2020/11/Logo.svg"
curl -o assets/logos/neo-logo-white.png "https://neopave.com/wp-content/uploads/2025/07/Asset-1.png"

# Fonts
curl -o assets/fonts/neo-thin.woff2 "https://neopave.com/wp-content/uploads/useanyfont/3996thin.woff2"
curl -o assets/fonts/neo-bold.woff2 "https://neopave.com/wp-content/uploads/useanyfont/6142bolder.woff2"

# Images
curl -o assets/images/neo-project-1.jpg "https://neopave.com/wp-content/uploads/2025/06/20201202_110952-Large.jpg"
curl -o assets/images/neo-project-2.jpg "https://neopave.com/wp-content/uploads/2025/06/neo-qatar1_IN03-scaled.jpg"
```

Once downloaded, the HTML has been updated to use these assets automatically.
